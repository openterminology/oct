# Mock Dataset Examples

This directory contains example FHIR $lookup responses for demonstration purposes.

## Available Examples

### SNOMED CT
- **44054006** - Type 2 diabetes mellitus
  - Multiple synonyms and designations
  - Parent relationship to Diabetes mellitus (73211009)
  - Module and effective time properties

### ICD-10
- **I10** - Essential (primary) hypertension
  - Chapter: Diseases of the circulatory system
  - Category: Hypertensive diseases
  - Clinical description included

### LOINC
- **718-7** - Hemoglobin [Mass/volume] in Blood
  - Long and short names
  - Component, Property, System, Scale Type
  - CLASS: HEM/BC (Hematology/Blood Count)

### RxNorm
- **313782** - Acetaminophen 325 MG Oral Tablet
  - Ingredient: Acetaminophen
  - Dose form: Oral Tablet
  - Strength: 325 MG
  - TTY (Term Type): SCD (Semantic Clinical Drug)

## Usage

These files demonstrate the variety of terminology systems supported by the OctoViewer:
- Different designation types (synonyms, preferred terms)
- Various property structures
- System-specific metadata
- Hierarchical relationships

⚠️ **Note:** This data is for demonstration purposes only and has not been clinically validated. Do not use for clinical decision-making.

# How to Attribute Open Content Terminology (OCT)

Use the following guidance to give clear, correct credit when you use OCT. Pick the block that matches your use case and include it in your documentation, about page, README, or citation list.

## If you use OCT software (MIT License)

Add this block to your project:

```
This project uses software from Open Content Terminology (OCT), licensed under the MIT License.
Copyright (c) 2024 Open Content Terminology (OCT) contributors. See https://github.com/OpenContentTerminology/oct for details.
```

## If you use OCT terminology or knowledge (CC BY 4.0)

Add this block to your materials:

```
Content includes terminology from Open Content Terminology (OCT), licensed under Creative Commons Attribution 4.0 International (CC BY 4.0).
Original source: https://github.com/OpenContentTerminology/oct. Changes (if any) are indicated in this work.
```

## If you use both software and terminology together

Add this combined block:

```
This work incorporates Open Content Terminology (OCT).
- Software components are licensed under the MIT License. Copyright (c) 2024 Open Content Terminology (OCT) contributors.
- Terminology and knowledge are licensed under Creative Commons Attribution 4.0 International (CC BY 4.0). Changes (if any) are indicated in this work.
Source: https://github.com/OpenContentTerminology/oct
```

## Tips for good attribution
- Keep the license names intact ("MIT License" and "Creative Commons Attribution 4.0 International").
- Link back to the OCT repository when possible.
- When you modify OCT terminology, briefly note the nature of your changes.
